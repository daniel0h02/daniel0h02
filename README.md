# 안녕하세요, 오찬경입니다 👋

AI Agent와 백엔드 개발을 중심으로 성장하고 있는 개발자입니다.  
사용자의 자연어 요청을 실제 서비스 기능으로 연결하고, 데이터 기반으로 신뢰할 수 있는 결과를 제공하는 개발에 관심이 있습니다.

특히 AI Agent, FastAPI, Spring Boot, PostgreSQL, Vector Search를 활용해  
자연어 기반 사원 검색과 사원 역량 분석 기능을 구현한 경험이 있습니다.
